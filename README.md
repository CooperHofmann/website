# Cooper of the Crop

A single-page photography studio website inspired by `cooperofthecrop.framer.photos`.

## Run locally

```bash
python -m http.server 8000
```

Open `http://localhost:8000`.

## Host for free

- **GitHub Pages**: push to GitHub, then enable Pages in repository settings (root folder).
- **Netlify**: drag-and-drop the folder or connect the GitHub repo.
- **Vercel**: import the repo and deploy as a static site.
